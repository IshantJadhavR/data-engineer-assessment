# Source Code

## SQL

- `sql/00_create_database.sql` creates the PostgreSQL database.
- `sql/01_create_tables.sql` creates the relational schema.
- `sql/02_load_data.sql` loads the CSV files.
- `sql/03_queries.sql` contains the join, aggregate, and window-function queries.

## Python

- `python/clean_song_plays.py` cleans the raw play data.
- `python/generate_sample_data.py` can recreate the synthetic sample CSVs.

## Data

- `data/raw/` contains the synthetic source CSVs.
- `data/cleaned/` contains the cleaned play data output.
